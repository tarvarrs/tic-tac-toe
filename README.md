# Tic-Tac-Toe

Проект Крестики-нолики представляет собой web-приложение на Go, следует 
чистой архитектуре и разделен на слои.

В приложении реализован алгоритм минимакс для поиска самого удачного хода 
для компьютера.

## 📂 Общая структура проекта

```bash
.
├── README.md
├── cmd
│   └── app          # Точка входа в приложение
├── go.mod
├── go.sum
└── internal
    ├── datasource   # Слой инфраструктуры для работы с данными
    ├── di           # Слой di с конфигурацией для внедрения зависимостей
    ├── domain       # Слой бизнес-логики приложения
    └── web          # Слой контрактов с API
```
